# LINE Patient Communication System

Built a LINE-based patient communication system using webhook automation, Flex Messages, and a structured Rich Menu.

## Impact
- Automated patient communication workflows via LINE OA  
- Improved structured interaction using 11 Flex Messages  
- Designed an 8-grid Rich Menu for fast navigation and usability  

## Overview
A LINE-based system that automates and structures patient communication workflows using webhook-driven interactions.

## Architecture
User → LINE OA → Webhook Server → Message Logic → LINE Flex / Rich Menu

## Tech Stack
- LINE Messaging API
- Webhook Automation
- Node.js
- JSON-based Flex Message structure
